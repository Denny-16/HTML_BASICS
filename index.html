IMPORTANT ELEMENTS IN HTML

HTML:

The root element of an HTML document. All other elements are nested within it

HEAD:

It contains meta-information about web page, such as pages title,links to external resoureces and character encoding declaration

TITLE:

Sets the title of the web page, which appears in the browser's title bar or tab

BODY:

Contains the visible content of web page,such as text,image,links, and multimedia

HTML-links

HTML links a to create hyperlinks that allows users to move from one web page to another
When a user clicks on a link,they are directed to URL specified in link's href attribute
links are essential for web navigation and hep users explore and interact with websites.

In this syntax
href: this attribute specifies URL or destination to which link should be pointed
target: this atttribute specifies how the linked page should open. In this _blank indicates the link should open in new browser tab or window
